悲伤日志


是这样的，我一开始上课没跟上，后来找教程打算用gitpage，看到很多人用jekyll弄现成主题就打算用这个了（现在想起就悔恨

然后需要安装ruby环境但是一直下不下来显示下载大小0b，教程里给的都只有官网链接，我以为是浏览器或者迅雷的问题，检查好久又没找到别的下载地址，后来在其他途径找到个大礼包成功下载才发现是官网给的无效

安装好了环境cmd输入ruby -v也有序列，再输入gem install jekyll 却告诉我ERROR: Could not find a valid gem 'jekyll' in any repository

继而发现是gem source里的网址被墙了，改成http才行，问Do you want to add this insecure source？[yn] 回y等了好久好久

我便继续输入gem install jekyll，又出现问题了 ERROR: Error installing jekyll:

 ERROR: Failed to build gem native extension.    好的继续google，告诉我是因为没装Devkit，输入个sudo apt-get install ruby-dev 就好了，现学现抄，cmd却告诉我不认识sudo， 问问google，啊，因为这是Linux的语言，我win不行，要装个Ubunto就好了

找教程又下了Ubunto和用来安装的VMware，到最后了出现Error 5，据称有三个原因，U盘或者内存条，但这些都太麻烦了我放弃了

这时候我想到还有Worldpress，于是去找教程，但我一直看不懂把文件上传到根目录啥意思，是要虚拟主机吗，hostinger虚拟主机说我注册过了免费的，我又不想再注册个gmail去注册hostinger。

这时候经同学提醒还有个mobirise，我心想我既然是数媒班的上过编程课有基础，我不能用傻瓜软件，于是我打开pencil想要自己排版原型，整了个主页导出了html的包上传gitpage，但一点变化都没有我茫然

最终我打开了mobirise，完成了现在这个网站，啊，好轻松好顺畅。这个过程太痛苦了，我现在愿意下辈子永远画画。
